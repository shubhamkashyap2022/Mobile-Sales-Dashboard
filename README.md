Mobile Sales Dashboard | Power BI
📌 Project Overview

This project is an interactive Mobile Sales Analytics Dashboard built in Microsoft Power BI using a Motorola mobile sales dataset.

The dashboard helps business users analyze sales performance, customer purchasing behavior, payment trends, and product performance across different cities and time periods.

The goal of this project is to transform raw sales data into actionable business insights using Power BI's visualization, data modeling, and DAX capabilities.
